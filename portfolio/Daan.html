Project Brief

Indians in general like to hold on to material objects and hoard them in the family home. This is the problem we are trying to tackle with The Daan Project. The community can now use a singular platform to source and donate any important artifacts or clothing that can be eventually used by the countless non for profits set up in India dedidcated to improving the quality of peoples life.

As a bonus the platform also sources local volunteering events geared towards the youth to introduce them into the practice of giving back to the society at a very early age..

Concept

The client contacted me with a request to design a website that allows two functions: Donations and Volunteering

After careful research and surfing through countless donation and volunteering websites, coupled with user research conducted to a very selected few early adopters, I was convinced the layout of the website has to revolve around the home page because there are only two major features the user is concerned about. By keeping those features prompty showcased across the banner, users are encourged to part take intuitively rather than on a forced or nudged basis. Additionally, the functions to donate and volunteer have to be segregated and they follow different interface designs to keep the user engaged across all the diffrent use cases.

Identity

The logo revolves around the global community coming together. The word "Daan" itself means to give back in Hindi. The arrow signifies the transfer from one side of the world to the other. When the animation is added, the goal is to have the globe slowly spinning on its axis. I also felt instead of using the saturated Samarkan font family, I did some research and found another Indian cultural infused font family that fits perfectly with the message conveyed by the platform.

Making a Donation

The user needs only three steps to make a donation. The total number of clicks from home - making a donation - back home is five.

Step 1 Select donate from the home screen. The user will have the option to access the button from three different call to actions - Banner, Landing page & Footer.

Step 2 The user is then prompted to select a type of donation. The options listed are: Books, Clothes and Custom donations.

Step 3 To specify the quantity the user has to play with the count carousel till they reach the desired number

Step 4 As there is no log in function for the platform, users have to input their information for the admin to reach out to them

Becoming a Volunteer

The second major use case for the platform is for the community to find and join volunteering events in their respective regions. The steps to join a cause are outlined below:

Step 1 Select Donation from any of the three call to actions on the home screen

Step 2 Select a cause that appeals to you. The causes have a thumbnail cover photo, description and how many volunteers still needed to reach their goal

Step 3 Once a cause is selected, a detailed description page will appear. The user can choose to view more causes or join this particular cause on this screen

Step 4 User is prompted to fill their information so the volunteering commitee can reach out to them and convey more information about the cause they have joined in

Adding a Cause

To keep the interface as similar as possible, the user can navigate to adding a cause through the main cause page. It is open ended so any user has the ability to add a cause. The interface design is exactly the same as the cause description screens.
